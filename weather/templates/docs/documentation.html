KnoxyView Weather — Documentation

Web dashboard that shows current weather, forecast, UV index and air quality (WAQI).

Quick Start (Windows)

  1. Create and activate a virtual environment: 
    python -m venv .venv
.\.venv\Scripts\Activate.ps1
  2. Install dependencies: 
    pip install -r requirements.txt
  3. Copy and edit environment variables: 
    copy .env.example .env
  4. Run migrations and start dev server: 
    python manage.py migrate
python manage.py runserver

Key UI Notes

  • Theme: Light/Dark toggle persists in localStorage.
  • AQI: WAQI is the primary source; provider AQI is used as a fallback.
  • Badges: Weather badges are generated server-side and shown under temperature.

Files of Interest

  • weather/views.py — data fetching and AQI logic
  • weather/templates/weather/index.html — main UI
  • docs/README.md — markdown version of this documentation
