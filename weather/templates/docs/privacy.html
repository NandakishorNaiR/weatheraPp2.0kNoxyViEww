Privacy Policy

Last updated: January 2026

1. Information We Collect

This application does not collect, store, or share any personal data.

  • Location information (city or place name) entered by the user to fetch weather data.
  • Technical data such as IP address and request logs, which may be automatically collected by the hosting provider (Render) for security and performance purposes.

2. How Information Is Used

Any location input is used only to retrieve weather information from a third-party weather API and is not stored.

3. Third-Party Services

This app uses third-party services to function properly:

  • Weather API services to fetch weather data.
  • Render (hosting provider) to host the application.

These services may collect limited technical information in accordance with their own privacy policies.

4. Data Storage

The application does not store user data in any database or persistent storage.

5. Cookies

This application does not use cookies for tracking or advertising purposes.

6. Data Security

Reasonable measures are taken to protect the application, but no method of transmission over the internet is 100% secure.

7. Children’s Privacy

This application is safe for general audiences and does not knowingly collect data from children under 13.

8. Changes to This Policy

This Privacy Policy may be updated in the future. Any changes will be reflected in this document.

9. Contact

If you have any questions about this Privacy Policy, please contact the repository owner via GitHub.
